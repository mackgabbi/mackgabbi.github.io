Welcome to the Official iMSMA Technology Club web page! Held here are links to tools that can answer any questions about website creation, information for the NJIT Web Design Competition, and features the works of the students of MSMA's Web Design Class!

Helpful Links

W3 Schools

W3 Schools is a great source for answering any questions you may have about coding or for searching for tags that may help improve your website!

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Codecademy

Codecademy is a great source for learning the basics of any programming language in a way that makes learning to program easy and enjoyable. From learning HTML to CSS and from Javascript to Ruby, Codecademy has it all!

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Hex Color Picker

This source is great for finding the perfect shade of any color that you may need to format your website. With the hex color system, you are not just limited to the 16 colors (red, blue, purple, orange, etc.) that you become familiar with when you first learn to code. With this website you have unlimited access to thousands of colors, all at your fingertips!

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Web Safe Fonts

This source is very useful for creating effective websites with fonts that are not just limited to a certain amount of computers. This page shows many fonts that are viewable on all computers that can make your website look professional.
